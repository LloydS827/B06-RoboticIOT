# B08 Equipment Timeseries Observation Package Profile

## Profile

| 项目 | 内容 |
| --- | --- |
| profile id | `equipment_timeseries_observation_package` |
| owning project | B08 设备时序基础模型 |
| 当前用途 | 定义设备时序观测包与 B06 Physical AI 数据层的衔接边界 |

## 定位

B08 profile 面向设备、传感器和工艺阶段的时序观测，不属于机器人 workcell 作业结构。它可以与 A01 或 S01 在事件、质量结果和候选窗口上发生引用关系，但不要求包含机器人 TCP、焊缝路径或相机位姿。

## 信息组

### 设备

- `equipment_id`、设备类型、设备组、产线或工站引用。
- 设备配置、版本、运行模式和维护状态。
- 设备身份若包含客户或现场敏感信息，应脱敏或仅保留 onsite-only 引用。

### 传感器

- `sensor_id`、传感器类型、采样频率、单位和量程。
- 数据来源、时间戳来源、缺失值和异常值规则。
- 传感器校准、替换或维护记录引用。

### 阶段

- 工艺阶段、运行阶段、状态切换或维护阶段。
- 阶段起止时间、来源系统和可信度。
- 阶段与质量结果、事件或模型评测窗口的绑定关系。

### Cycle

- `cycle_id`、批次、班次或作业周期引用。
- cycle 内的开始、稳定、结束、停机或异常片段。
- cycle 与设备、工艺、质量标记的关系。

### 窗口

- 可复盘的时间窗口，例如异常前后、模型候选片段或质量波动片段。
- 窗口起止时间、窗口长度、重采样策略和来源引用。
- 与 B06 candidate sample export 或 replay artifact 的关系。

### 质量标记

- 合格/不合格、缺陷、报警、返修、停机或人工复核结果。
- 标签来源、reviewer/source 和质量系统引用。
- 标签与设备、sensor、cycle、窗口之间的绑定关系。

### 模型评测

- 模型版本、输入窗口、预测目标和评测指标。
- 预测结果、误差、置信度和失败样本引用。
- 评测结果可以作为 B06 派生结果引用，但不替代原始观测。

### 候选信号

- 与故障、质量波动、能耗、工艺偏移或维护相关的候选 signal。
- signal 来源、筛选规则、时间窗口和人工确认状态。
- 未确认候选只作为 context，不作为最终标签。

## 与 B06 的关系

- B06 提供共享的数据治理口径：Raw/Clean/Package、脱敏、不可提交数据和 artifact 边界。
- B06 可以保留可复盘窗口、候选样本、质量结果引用和模型评测摘要。
- B06 不把 B08 profile 塞进 `weld_workcell_job_window`，也不要求 B08 产生机器人轨迹字段。

## 非目标

- 不实现设备生产 connector。
- 不定义 B08 长期训练数据 schema。
- 不要求 B08 数据具备机器人 workcell 坐标系。
- 不提交未脱敏现场设备时序或权限配置。
