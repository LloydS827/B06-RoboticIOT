# A02 Manipulation Skill Asset Evidence Profile

## Profile

| 项目 | 内容 |
| --- | --- |
| profile id | `manipulation_skill_asset_evidence` |
| owning project | A02 机器人技能大师 |
| 关联对象 | `ManipulationSkillAsset` |
| 当前用途 | 定义 B06/A01 可交给 A02 的技能资产 evidence 边界 |

## 与 ManipulationSkillAsset 的关系

本 profile 不是 A02 `ManipulationSkillAsset` 的完整 schema，而是 B06 向 A02 交付 evidence 时的前置 contract。它说明哪些 A01 H300 作业窗口字段可以支持技能资产创建、审查、迁移评测或失败复盘。

原始 H300 数据不会自动成为技能资产。只有经过脱敏、来源可追溯、上下文清楚，并且通过人工确认或质量结果支撑的片段，才适合作为 `ManipulationSkillAsset` 的候选 evidence。

## Evidence 信息组

### 确认轨迹、TCP 和路径点

- 经人工确认或质量结果支撑的 TCP 轨迹片段。
- 规划路径、执行路径和人工修正路径的对照。
- 路径点所属阶段、焊缝、工件和坐标系。
- 轨迹来源引用，例如 Clean Zone 文件、Package artifact 或 onsite-only source ref。

### 上下文与技能意图

- 操作目标，例如接近焊缝、跟踪焊缝、避障、退枪或复位。
- 工件、焊缝、工具、机器人和工站上下文。
- 坐标系、标定版本、tool frame 和 TCP 假设。
- 与技能泛化有关的材料、几何或姿态条件。

### 质量标签

- 合格/不合格、缺陷类型、返修、外观或尺寸结果。
- 标签来源：专家审查、检测系统、抽检记录或质量系统摘要。
- 标签与轨迹片段、焊缝和时间窗口的绑定关系。

### 专家审查

- 专家确认的轨迹是否可复用。
- 人工修正的原因、修正方式和可信度。
- reviewer/source 的脱敏引用。
- 审查结论是否允许进入技能资产候选集。

### 迁移评测线索

- 可用于相似工件、相似焊缝或相似姿态的条件描述。
- 需要重新标定、重新示教或重新评估的触发条件。
- 与 A02 评测任务相关的候选指标，例如路径偏差、接近姿态、失败阶段和质量结果。

### 失败边界

- 失败类型、失败阶段和影响对象。
- 异常原因、报警或人工复盘摘要。
- 不适合复用为技能的边界，例如标定失效、工件偏差过大、点云缺失或模型输出未确认。

## 只能作为上下文或附件的内容

- 大体量原始点云、图像、日志、PCL 中间文件。
- 现场敏感工单、客户标识、设备身份、人员信息或未脱敏原始文件。
- 未经确认的模型中间输出。
- 无法追溯时间、坐标系或质量结果的轨迹片段。

## 非目标

- 不要求 A02 直接消费完整 A01 作业包。
- 不把所有 H300 原始数据转成技能资产。
- 不定义 A02 训练 pipeline、评测平台或资产库实现。
- 不复制不可提交数据；必要时只保留 onsite-only source reference。
